## 六、常见问题

### Q：免费期过了怎么办？

无影云免费期1个月到期后：
- **方案A**：换一个手机号重新注册（最省钱）
- **方案B**：买阿里云ECS最便宜的实例（约50元/月）
- **方案C**：用本地电脑跑（但需要一直开着）

### Q：MiMo体验金用完了怎么办？

- 充值很便宜，几毛钱一天
- 也可以去其他平台薅免费额度（硅基流动、DeepSeek等都有新人免费额度）
- 体验金40天有效，个人日常使用足够

### Q：AI回复很慢怎么办？

- 检查云电脑配置是不是太低（4核8G够用，2核4G可能卡）
- 换一个更快的模型（MiMo-v2.5通常比DeepSeek快）
- 检查网络：`ping api.xiaomimimo.com`

### Q：微信对接总是失败？

- 确认你的云电脑能正常访问外网
- 检查Hermes版本是否最新：`hermes --version`
- 重新走一遍第五步的对接流程，注意每一步的提示信息

### Q：免费期过后怎么续用？

每天签到领无影豆，攒够了续用：

打开无影APP → 底部点「我的」→ 点「签到」。

- 每天签到必得 **10个无影豆**

- 一个月签满 ≈ 480个无影豆，够免费续用云电脑

## 七、我的踩坑记录

这是我从2026年3月到7月，用这套方法搭系统的真实踩坑经历。你大概率也会遇到。

### 坑1：无影云到期前没备份，数据全丢

**发生了什么**：免费期过了，阿里云自动释放了云电脑，我桌面上的文件全没了。

**怎么解决**：养成习惯，每周备份一次到手机或GitHub。

```bash
# 备份命令
tar -czf ~/backup-$(date +%Y%m%d).tar.gz ~/Desktop ~/Documents
```

### 坑2：API密钥泄露

**发生了什么**：我把密钥贴到了一个公开的GitHub仓库里，被人薅了几十块钱的API调用。

**怎么解决**：密钥永远不要提交到GitHub。用环境变量管理：

```bash
# 在 ~/.bashrc 里写
export MIMO_API_KEY="sk-你的密钥"
```

然后代码里读环境变量，不硬编码密钥。

### 坑3：Hermes配置文件格式错误

**发生了什么**：我手动编辑config.yaml，少了一个缩进，Hermes启动就报错。

**怎么解决**：用命令行改配置，不要手动编辑文件：

```bash
hermes config set xxx yyy
```
